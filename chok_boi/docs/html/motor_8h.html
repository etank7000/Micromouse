void setLeftMotor ( int32_t  val)

Set the duty cycle for the left motor.

The argument takes in an integer ranging from -999 to 999, where 0 is effectively a 0% duty cycle and 999 is effectively a 100% duty cycle. A positive value will spin the motor forward, and a negative value will spin the motor backward.
